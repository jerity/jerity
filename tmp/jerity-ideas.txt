=== Handle Special Metadata Links ===
 - Add helper function to generate link tags.
 - Specification:
  * http://www.w3.org/TR/html401/struct/links.html
  * http://www.w3.org/TR/html401/struct/links.html#search
 - Available link types:
  i. '''alternate''' - An alternate version of the document (i.e. print page, translated or mirror)
  i. '''stylesheet''' - An external style sheet for the document
  i. '''start''' - The first document in a selection
  i. '''next''' - The next document in a selection
  i. '''prev''' - The previous document in a selection
  i. '''contents''' - A table of contents for the document
  i. '''index''' - An index for the document
  i. '''glossary''' - A glossary (explanation) of words used in the document
  i. '''copyright''' - A document containing copyright information
  i. '''chapter''' - A chapter of the document
  i. '''section''' - A section of the document
  i. '''subsection''' - A subsection of the document
  i. '''appendix''' - An appendix for the document
  i. '''help''' - A help document
  i. '''bookmark''' - A related document
 - Other link types:
  i. '''home''' - The home page of the site
 - Available media descriptors:
  i. '''screen''' - Intended for non-paged computer screens.
  i. '''tty''' - Intended for media using a fixed-pitch character grid, such as teletypes, terminals, or portable devices with limited display capabilities.
  i. '''tv''' - Intended for television-type devices (low resolution, color, limited scrollability).
  i. '''projection''' - Intended for projectors.
  i. '''handheld''' - Intended for handheld devices (small screen, monochrome, bitmapped graphics, limited bandwidth).
  i. '''print''' - Intended for paged, opaque material and for documents viewed on screen in print preview mode.
  i. '''braille''' - Intended for braille tactile feedback devices.
  i. '''aural''' - Intended for speech synthesizers.
  i. '''all''' - Suitable for all devices.
 - Examples:
  i. `<link rel="home" href="/" />`
  i. `<link rel="copyright" href="#copyright" />`
  i. `<link rel='index' title='Mozilla Hacks - the Web developer blog' href='http://hacks.mozilla.org' />`
  i. `<link rel='start' title='Firefox 3.5 for 35 days &#8211; dreaming about the future of the web' href='http://hacks.mozilla.org/2009/06/35-days/' />`
  i. `<link rel='prev' title='Mozilla developer preview (Gecko 1.9.3a1) available for download' href='http://hacks.mozilla.org/2010/02/mozilla-developer-preview-gecko-1-9-3a1-available-for-download/' />`
  i. `<link rel='next' title='improving JavaScript performance with JÃ¤gerMonkey' href='http://hacks.mozilla.org/2010/03/improving-javascript-performance-with-jagermonkey/' />`

 - Apple Touch Support
  * Specification: http://developer.apple.com/safari/library/documentation/AppleApplications/Reference/SafariWebContent/ConfiguringWebApplications/ConfiguringWebApplications.html
  * (X)HTML Meta Tag:
   - `<meta name="apple-mobile-web-app-capable" content="yes" />`
   - `<meta name="apple-mobile-web-app-status-bar-style" content="black" />`
  * (X)HTML Link Tag:
   - `<link rel="apple-touch-icon" href="/images/apple-touch-icon.png"/>`
   - `<link rel="apple-touch-icon-precomposed" href="/images/apple-touch-icon.png"/>`
   - `<link rel="apple-touch-startup-image" href="/startup.png">`
 - Android Support
  * The option ''apple-touch-icon-precomposed'' (see above) is supported.
